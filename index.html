Dúvidas frequentes

Respondendo suas perguntas

Precisa de internet para usar o sistema? +

Não! O NEXUS é um arquivo HTML que funciona direto no seu navegador (Chrome, Firefox, Edge). Você só precisa de internet na primeira abertura para carregar as fontes visuais. Depois, funciona offline normalmente.

É necessário instalar algum programa? +

Não precisa instalar nada. Basta baixar o arquivo e abrir com qualquer navegador moderno. Simples assim.

Meus dados ficam salvos? +

Os dados que você lança ficam visíveis enquanto o arquivo está aberto. O sistema não tem banco de dados online, por isso recomendamos não fechar o navegador sem antes anotar informações importantes. Uma boa prática é usar junto com uma planilha de backup.

Como funciona a garantia de 7 dias? +

Se por qualquer motivo você não ficar satisfeito nos primeiros 7 dias após a compra, basta entrar em contato e devolvemos 100% do valor pago. Sem perguntas, sem burocracia.

Funciona no celular? +

O sistema foi desenvolvido para funcionar melhor no computador (desktop/notebook). No celular pode abrir, mas a experiência é mais limitada por conta do tamanho da tela.

O que é o vídeo tutorial? +

É um vídeo gravado mostrando como usar cada módulo do sistema: como lançar receitas e despesas, cadastrar clientes, criar metas, usar a agenda e exportar relatórios em PDF. Tudo passo a passo para você dominar o NEXUS rapidinho.
